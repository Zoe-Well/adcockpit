## 职位详情
我们是抖音、腾讯视频号、快手、小红书等头部平台的官方授权服务商，为2B企业客户提供从广告投放、内容生产、电商闭环到数据分析的全链路数字营销服务。
## 岗位职责
负责数字营销 AI Agent System 的架构设计与开发，包括 Multi - Agent 编排框架、工具调用链路、记忆与状态管理等核心模块。
深入理解数字营销业务场景（广告投放、内容生产、电商运营、数据分析），将业务需求转化为 Agent 能力设计。
基于 AI Coding 工作流（Claude Code / Codex 等），快速开发和迭代各业务线 Agent，实现端到端的自动化工作流。
构建 Agent 的工具生态（Tool/Function Calling），打通各平台 API（巨量引擎、磁力引擎、腾讯广告、小红书聚光等）。
## 任职要求
学历不限，计算机、AI、软件工程等相关专业优先，具备基本的编程能力。
熟练使用 AI Coding 工具（Claude Code / Codex 等），能用 AI 完成 95% 以上的编码工作，这是我们团队的核心工作方式。
了解 LLM 基本原理（Prompt Engineering、RAG、Function Calling），有使用或搭建 Agent 的实际经验（哪怕是个人项目或 Demo 也算）。
熟悉 OpenClaw 等 AI Agent 开发平台，熟悉至少一种 Agent 开发框架（LangChain / LangGraph / CrewAI 等），能快速搭建和迭代 Agent 工作流。
有数字营销 / 广告投放 / 电商行业经验，理解巨量引擎、腾讯广告等平台的业务逻辑优先。
