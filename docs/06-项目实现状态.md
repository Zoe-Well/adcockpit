## 一、架构概览

```
React 18 + Tailwind CSS + shadcn/ui (5173)  ←→  FastAPI (8000)  ←→  Mock Functions + Feishu Client
                                                                  ←→  Local JSON (backend/data/)
```

| 层 | 技术 | 状态 |
|----|------|:--:|
| 前端 | React 18 + TypeScript + Tailwind CSS + shadcn/ui 组件 | ✅ |
| 后端 | FastAPI + Pydantic | ✅ |
| Agent 编排 | LangGraph (代码已迁移, 未接入 LLM) | ⚠️ |
